This is an interactive Markowitz portfolio optimizer built as a browser-based tool that runs entirely without installation. Users can search any stock ticker or company name, select up to 15 stocks from a universe spanning US equities, Canadian TSX listings, and global ADRs, and run a mean-variance optimization to find the minimum-risk portfolio allocation. The app fetches two years of live price data from Yahoo Finance, computes the full covariance matrix, runs a gradient-descent optimization to trace the efficient frontier across 22 target return points, and surfaces the results as an interactive chart alongside a weighted holdings table with direct links to Yahoo Finance and MarketWatch.

I built this to show how I work at the intersection of quantitative modeling and product thinking. The mathematical foundation comes directly from my MMA coursework in prescriptive analytics and machine learning, translated from a Jupyter notebook into something anyone can use in a browser in under a minute. I used Claude to accelerate the frontend scaffolding while driving all modeling decisions, UX choices, and output framing myself. The result is a tool that makes a graduate-level finance concept immediately tangible, which I think reflects how I approach data science work more broadly: the model only matters if someone can actually use it.
